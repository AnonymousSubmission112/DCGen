Features

  • Supports any doc tool
    Integrated support for Sphinx, MkDocs, Jupyter Book. Flexible enough to work with any documentation tool.
  • Integrated search
    Easily search all your docs and other databases. And you'll get a free simple search with no need to set up.
  • Pull request previews
    Review each pull request before merging any changes, allowing for real collaboration across your organization.
  • Versioned docs
    Match your project release cycle with multiple versions of your docs. Automatically build versions from any branch or tag.
  • Private repository support
    Clone private repositories using a connected GitHub, GitLab, or Bitbucket account. Or clone from any Git provider with a HTTPS URL.
  • Single-Sign On
    For your whole team.
    Integrate with systems to your documentation with integrated authentication. Enable SSO with GitHub or Google to easily provision access for your team.
