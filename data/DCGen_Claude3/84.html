Frequently Asked Questions

How often do you update your database?

We update our database as soon as we have new information about an IP block. For easy API request flow, all clients have the most accurate location data without having to worry about updating a local database.

How many requests can I do?

Our requests are limited to 5M HTTP requests per month from an IP address. If you go over this limit, your requests will be throttled HTTP 429 until your rate limit window is reset.

What is the average response time of the API?

With dedicated servers in US, EU and APAC, a network based on Anycast technology, and highly optimized software, we achieve real response times of under 20 milliseconds in most parts of the world.

Can I use your API on my commercial website?

We do not allow commercial use of the free endpoint. Please see our pricing page for SSL, domain, website domains, image statistics and commercial support.

Do I need an API key for the free endpoint?

Yes all users require an API key or integration and the API is metered will not charge.
