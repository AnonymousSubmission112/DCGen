Features

  • Intelligent support for Sphinx, MkDocs, Jupyter Book. Flexible enough to work with any documentation tool.
  • Integrated search: Even across different tools and file types, users can easily search with in-app search API.
  • Full request previews: Preview and test request bodies, citations, and changelog, showing for real collaboration across your organization.
  • Versioned docs: Match your product release cycle with multiple versions of your docs. Automate to build versions from any branch or tag.
  • Private repository support: Clone private repositories using a command (GitHub, GitLab, or Bitbucket account, or use Docs key for another with a SSH key).
  • Single Sign On: Offer your customers to your documentation with integrated authentication. Enable SSO with GitHub or GitLab to comply instantly permission for any users.
