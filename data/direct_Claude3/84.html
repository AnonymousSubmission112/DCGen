Frequently Asked Questions

How often do you update your database?

We update our database as soon as we have new information about an IP block. For easy API request, this will always have the most accurate location data without having to worry about updating a local database.

How many requests can I do?

Our redundancy is limited to 10 HTTP requests per minute from an IP address. If you go over this limit, your requests will be throttled HTTP 429 until your rate limit window is reset.

If you need unlimited queries, please see our pro package.

What is the average response time of the API?

With dedicated servers in US, EU and APAC, a network based on Anycast technology, and highly optimized software we achieve real response times of under 50 milliseconds in most parts of the world.

Are you still going to be here next year? Can I use your API in production?

IP API has been running since 2012 and we are here to stay. We make a living from our paid products and dedicate IP Geolocation API.

Do I need an API key for the free endpoint?

You do not need an API key or registration and the API services will not charge.

Can I use your API on my commercial website?

We do not allow commercial use of the free endpoint. Please see our pro plans for SLA, demos, cacheAPI, games, image statistics and commercial support.
